# 🧠 Project DNA

> **Version:** 1.0.0 | **Last Updated:** 2026-01-28 | **Status:** 🟢 Active

---

## 🎯 Mission Statement

> สร้าง **Web Application** เป็นศูนย์กลางจัดการโครงงานนักศึกษา เชื่อม **Pre-Project → Project** แบบต่อเนื่อง พร้อมติดตาม ประเมิน และจัดเก็บโครงงานอย่างเป็นระบบ

---

## 💎 Core Values

| Value                    | Description                                   |
| ------------------------ | --------------------------------------------- |
| 🔗 **Continuity**        | Pre-Project และ Project เชื่อมต่อกันไม่ขาดตอน |
| 👥 **Role-Clarity**      | แต่ละบทบาทมีหน้าที่ชัดเจน ไม่ทับซ้อน          |
| 📊 **Transparency**      | ติดตามความคืบหน้าได้ตลอดเวลา                  |
| 📚 **Knowledge Archive** | เก็บองค์ความรู้สำหรับรุ่นต่อไป                |

---

## 🎯 Project Goals

1. ✅ จัดการโครงงานนักศึกษาแบบ Pre-Project และ Project
2. ✅ ติดตามความคืบหน้าของงานแบบ Real-time
3. ✅ รองรับการทำงานเป็นทีมและการมอบหมายงาน
4. ✅ ระบบประเมินผลและให้คะแนน
5. ✅ การแจ้งเตือนและการสื่อสารภายในทีม
6. ✅ รองรับการเข้าถึงด้วยบทบาทต่างๆ (Role-Based Access)

---

## 👥 Target Users

| Role           | Description             |
| -------------- | ----------------------- |
| 👨‍🎓 **Student** | นักศึกษาที่ทำโครงงาน    |
| 👨‍🏫 **Advisor** | อาจารย์ที่ปรึกษาโครงงาน |
| 👨‍💼 **Admin**   | ผู้ดูแลระบบ             |
